# v3.5.0(Sept 27, 2022)

### New Features
* Windows平台支持64位版本
* 设置界面增加长按空格解除静音功能开关
* 支持开启白板后，可通过点击日志按钮上传日志

### Bug Fixes
* 解决在移动端取消授权后，桌面端依然有权限的问题
* 解决开启音频入会后，音频状态显示不对的问题

### Compatibility
* Compatible with `NEMeetingKit` version `3.5.0`
