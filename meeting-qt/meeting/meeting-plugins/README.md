# NetEase Meeting Qt Quick Plugin

网易会议基于 Qt Quick 插件的封装，您可以通过该插件快速创建一个实时音视频通讯的应用，如在线教育、视频会议等。

## 最低版本要求

| 工具 | 版本 |
| -- | -- |
| Qt | 5.14.2 |
| Visual Studio | 2017 |
